## Nesting / Descendant Element

2. I used (main p) which styles the paragraphs that is only found inside <main>. I set its background-color to #A4B7A9; (very light green) and made their font-size to take 90% of the parent(100%).

3. I used (main dl dt) which styles the dt elements only inside the dl which can only be found in <main> and no where else. I set its font-size to 16px and its foreground color to red.

4. I used (main article) to style articles that are found inside <main>. I set their font-size to 95%.

5. I used (main article h4) to style the headings inside articles which is inside <main>. I added a text-decoration to underline it.

    ## ID Selectors:

6. I used #main-navigation to style the main navigation bar. I set its background-color to  #5A7D7C; (Teal color).

7. I used #main-navigation a to style the links <a> that is inside the navigation bar. I set its foreground color to white.

    ## Class Selectors: 

8. I used .footer-content p to style the <p> paragraph inside the footer container. I set its font-size to 15px
